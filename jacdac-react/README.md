## JACDAC on React with TypeScript

This is a showcase app that allows you to browse JACDAC devices and its services in an interactive way. The app is built with React and Bootstrap.

### Setup

Requires Node.js:

```
npm install
```

### Start Dev Server

```
npm start
```

### Serve Production

```
npm run build
npm i -g http-server
http-server ./build
```
